# SimpleCalculator Project

## Description
The SimpleCalculator project is a Python-based application that performs basic arithmetic operations such as addition, subtraction, multiplication, and division. It is designed to provide a straightforward interface for users to perform calculations easily.

## Features
- Supports addition, subtraction, multiplication, and division.
- User-friendly input prompts.
- Handles invalid inputs gracefully.
- Displays operation results clearly.

## Usage Instructions
1. Make sure you have Python installed on your machine.
2. Clone the repository:
   ```
git clone https://github.com/nasrin-hosini/my-first-p-t1.git
```
3. Navigate to the project directory:
   ```
cd my-first-p-t1
```
4. Run the SimpleCalculator:
   ```
python py.py
```

## Examples
- Addition: Input `5 + 3` will return `8`.
- Subtraction: Input `10 - 4` will return `6`.
- Multiplication: Input `7 * 2` will return `14`.
- Division: Input `20 / 5` will return `4`.
